# Paradise Bounty

A fun little mod that adds more end game stuff to Paradise Lost! 
Later on in a modded world, you lose the surtrum and glazed gold tools, so this mod gives you a reason to come back and do more!
Very small project, please support the original devs :3

## DEPENDENCY
Please add Paradise Lost!
